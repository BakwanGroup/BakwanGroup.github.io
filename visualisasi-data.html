Sumber Data dan Metode Pengambilannya

Sumber data berupa destinasi wisata sebanyak 829 berasal dari provinsi 7 yaitu Jawa Barat, Jawa Timur, Jawa Tengah, Banten, Bali, Jakarta, dan Yogyakarta, dengan total 75 Kabupaten/Kota dari ke-7 provinsi tersebut. Web scraping dilakukan pada Wikipedia "Daftar tempat wisata di Indonesia" menggunakan library BeautifulSoup. Hasil scraping berupa nama tempat wisata berserta letak kabupaten/kota dan provinsi tempat wisata tersebut berada.

Hasil scraping nama-nama tempat destinasi ini kami gunakan sebagai acuan ketika scraping data gambar berupa foto. Total gambar yang kamu dapatkan berjumlah 800 file yang diambil dari thumbnail pertama pada hasil pencarian di Google Images. Proses scraping dilakukan dengan script yang kami buat menggunakan library Selenium.

Untuk melengkapi informasi tempat-tempat wisata pada web kami, kami melakukan web scraping pada Google Maps untuk mendapatkan informasi tambahan berupa alamat lengkap, nomor telepon, website resmi, rating, serta letak tempat wisata berdasarkan google maps.

Metode yang kami gunakan dalam proses pengambilan data yaitu metode Web Scraping. Data kumpulan destinasi wisata kami dapatkan dari website Wikipedia. Gambar didapat dari pencarian Google Images, sedangkan data Google Maps didapat dari Scraping menggunakan tool APIFY.
